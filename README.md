# About
  A simple Proof-Of-Work system building with Go and Vue 3   

# How to run it
##  backend :  
  1 cd to the backend folder   
  2 `go run server.go`  
##  frontend :  
  1 cd to the frontend folder  
  2`npm install`  
  3`npm run dev`  
If you have any question feel free to solve it yourself
